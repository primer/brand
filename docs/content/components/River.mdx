### Image to text ratio

```jsx live
<Container p={3}>
  {/* 50/50 (default) example */}
  <River>
    <River.Visual>
      <img
        src="https://via.placeholder.com/600x400/f5f5f5/f5f5f5.png"
        alt="placeholder, blank area with an off-white background color"
      />
    </River.Visual>
    <River.Content>
      <Heading>50/50</Heading>
      <Text>
        <strong>By default</strong>, River applies a 50/50 image to text split.
      </Text>
    </River.Content>
  </River>
  {/* 60/40 example */}
  <River imageTextRatio="60:40">
    <River.Visual>
      <img
        src="https://via.placeholder.com/600x400/f5f5f5/f5f5f5.png"
        alt="placeholder, blank area with an off-white background color"
      />
    </River.Visual>
    <River.Content>
      <Heading>60/40</Heading>
      <Text>This example applies an optional 60/40 image to text split.</Text>
    </River.Content>
  </River>
</Container>
```

## Component props

### River <Label>Required</Label>

| Name             | Type                            |   Default   | Description                                                                |
| :--------------- | :------------------------------ | :---------: | :------------------------------------------------------------------------- |
| `align`          | `'left'`, `'right'`, `'center'` |  `'left'`   | Alignment of text content relative to the Visual position                  |
| `imageTextRatio` | `'50:50'`, `'60:40'`            |  `'60:40'`  | The aspect ratio applied to the image. Affects overall layout proportions. |
| `as`             | `'section'`, `'div' `           | `'section'` | HTML tag to be used in the parent element                                  |

### River.Visual <Label>Required</Label>

| Name       | Type           | Default | Description                                                                           |
| :--------- | :------------- | :-----: | :------------------------------------------------------------------------------------ |
| `children` | `ReactElement` |         | Bring your own component (BYOC) `img` or `ReactNode` (E.g. Next.js `Image` component) |

### River.Content <Label>Required</Label>

| Name       | Type                                          | Default | Description |
| :--------- | :-------------------------------------------- | :-----: | :---------- |
| `children` | `Text`,`Heading`, `LinkButton`, `River.Quote` |         |             |
