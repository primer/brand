## Component props

### River <Label>Required</Label>

| Name         | Type                            |   Default   | Description                                                                |
| :----------- | :------------------------------ | :---------: | :------------------------------------------------------------------------- |
| `alignment`  | `'left'`, `'right'`, `'center'` |  `'left'`   | Alignment of text content relative to the Visual position                  |
| `imageRatio` | `'50/50'`, `'60/40'`            |  `'60/40'`  | The aspect ratio applied to the image. Affects overall layout proportions. |
| `as`         | `'section'`, `'div' `           | `'section'` | HTML tag to be used in the parent element                                  |

### River.Visual <Label>Required</Label>

| Name       | Type           | Default | Description                                                                     |
| :--------- | :------------- | :-----: | :------------------------------------------------------------------------------ |
| `children` | `ReactElement` |         | Bring your own component (BYOC) img or ReactNode (E.g. Next.js Image component) |

### River.Content <Label>Required</Label>

| Name       | Type                                                         | Default | Description |
| :--------- | :----------------------------------------------------------- | :-----: | :---------- |
| `children` | `River.Heading`, `River.Body`, `River.Button`, `River.Quote` |         |             |

### River.Heading

| Name       | Type                                        | Default | Description                                                |
| :--------- | :------------------------------------------ | :-----: | :--------------------------------------------------------- |
| `as`       | `'h1'`, `'h2'`,`'h3'`,`'h4'`,`'h5'`,`'h6'`, | `'h3'`, | Optionally adjust heading size and underlying HTML element |
| `children` | `string`,                                   |         | Heading text                                               |

### River.Body <Label>Required</Label>

| Name       | Type      | Default | Description    |
| :--------- | :-------- | :-----: | :------------- |
| `children` | `string`, |         | Main body text |

### River.Button

| Name       | Type              | Default | Description                    |
| :--------- | :---------------- | :-----: | :----------------------------- |
| `as`       | `'button'`, `'a'` |  `'a'`  | Optionally adjust heading size |
| `children` | `string`,         |         | Button text                    |

### River.Quote.Body

| Name       | Type     | Default | Description                    |
| :--------- | :------- | :-----: | :----------------------------- |
| `children` | `string` |         | Optionally adjust heading size |

### River.Quote.From

| Name       | Type     | Default | Description                    |
| :--------- | :------- | :-----: | :----------------------------- |
| `children` | `string` |         | Optionally adjust heading size |
