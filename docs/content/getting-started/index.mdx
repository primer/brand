Primer React Brand is [open-sourced on GitHub](https://github.com/primer/react-brand) and [available on npm](https://www.npmjs.com/package/@primer/react-brand).

### 1. Install Primer React Brand

Install Primer React Brand with [npm](https://www.npmjs.com/).

```shell
npm install @primer/react-brand
```

### 2. Import global stylesheets

Primer React Brand requires a global stylesheet loaded ahead-of-time.

#### a. Using an application bundler (e.g. Webpack)

Import the stylesheet at the earliest React-rendering opportunity:

```js
import '@primer/react-brand/lib/css/gh-variables.color.css'
import '@primer/react-brand/lib/css/main.css'
```

or

#### b. Using a static stylesheet

Use a static stylesheet if you're unable to load in global styles using a bundler

```html
<link
  href="https://unpkg.com/browse/@primer/react-brand@<version>/lib/css/main.css"
  rel="stylesheet"
/>
<link
  href="https://unpkg.com/browse/@primer/react-brand@<version>/lib/css/gh-variables.color.css"
  rel="stylesheet"
/>
```

### 3. Import components

Components can be imported into any React-based application.

E.g.

```js
import {Hero} from '@primer/react-brand'
```

```jsx
<Hero {...props} />
```
