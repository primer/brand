## Examples

### Default

The default card displays a heading and description. The link text defaults to `Read more`, but can be customized using the `label` prop.

```jsx live
<Card href="https://github.com">
  <Card.Heading>Collaboration is the key to DevOps success</Card.Heading>
  <Card.Description>
    Everything you need to know about getting started with GitHub Actions.
  </Card.Description>
</Card>
```

### Link text

The link text defaults to `Read more` and can be customized using the `label` prop.

```jsx live
<Card label="Click here" href="https://github.com">
  <Card.Heading>Collaboration is the key to DevOps success</Card.Heading>
  <Card.Description>
    Everything you need to know about getting started with GitHub Actions.
  </Card.Description>
</Card>
```

### Label

Use the `Label` to add metadata to the `Card`. The `Label` is optional and can be used to indicate the status of the content.

```jsx live
<Card href="https://github.com">
  <Card.Label>Limited</Card.Label>
  <Card.Heading>Code search & code view</Card.Heading>
  <Card.Description>
    Enables you to rapidly search, navigate, and understand code, right from
    GitHub.com.
  </Card.Description>
</Card>
```

#### Color customization

The default colors can be customized through a dedicated CSS variable.

| Name                       | Default                      |
| :------------------------- | :--------------------------- |
| `--brand-Card-color-label` | `--brand-color-text-default` |

```javascript live noinline
const App = () => {
  const designTokenOverrides = `
  .custom-colors {

    /*
     * Modify the value of these tokens.
     */
    --brand-Card-color-label: var(--base-color-scale-purple-2);
  }
`

  return (
    <ThemeProvider colorMode="dark">
      <style>{designTokenOverrides}</style>
```

### Stacked cards

Use the `Stack` component to stack multiple cards horizontally or vertically.

```jsx live
<Stack
  direction="horizontal"
  alignItems="center"
  justifyContent="center"
  gap="normal"
>
  <Card href="https://github.com">
    <Card.Heading>Heading</Card.Heading>
    <Card.Description>
      Everything you need to know about getting started with GitHub Actions.
    </Card.Description>
  </Card>
  <Card href="https://github.com">
    <Card.Heading>Heading</Card.Heading>
    <Card.Description>
      Everything you need to know about getting started with GitHub Actions.
    </Card.Description>
  </Card>
  <Card href="https://github.com">
    <Card.Heading>Heading</Card.Heading>
    <Card.Description>
      Everything you need to know about getting started with GitHub Actions.
    </Card.Description>
  </Card>
</Stack>
```

## Component props

### Card

| name       | type     | default     | required | description                                 |
| ---------- | -------- | ----------- | -------- | ------------------------------------------- |
| `href`     | `string` |             | `true`   | URL to the card content                     |
| `linkText` | `string` | `Read more` | `false`  | Label of the link at the bottom of the Card |

### Card.Heading

| name | type                         | default | required | description |
| ---- | ---------------------------- | ------- | -------- | ----------- |
| `as` | `h2`, `h3`, `h4`, `h5`, `h6` | `h3`    | `false`  |             |
