## Examples

### Default

The default card can be built with a custom heading and a description. The link label defaults to `Read more` and can be customized using the `link` prop.

```jsx live
<Card linkHref="https://github.com">
  <Card.Heading>Collaboration is the key to DevOps success</Card.Heading>
  <Card.Description>
    Everything you need to know about getting started with GitHub Actions.
  </Card.Description>
</Card>
```

### Link

The link label defaults to `Read more` and can be customized using the `link` prop.

```jsx live
<Card link="Click here" linkHref="https://github.com">
  <Card.Heading>Collaboration is the key to DevOps success</Card.Heading>
  <Card.Description>
    Everything you need to know about getting started with GitHub Actions.
  </Card.Description>
</Card>
```

### Stacked cards

Use the `Stack` component to stack multiple cards horizontally or vertically.

```jsx live
<Stack
  direction="horizontal"
  alignItems="center"
  justifyContent="center"
  gap="normal"
>
  <Card linkHref="https://github.com">
    <Card.Heading>Heading</Card.Heading>
    <Card.Description>
      Everything you need to know about getting started with GitHub Actions.
    </Card.Description>
  </Card>
  <Card linkHref="https://github.com">
    <Card.Heading>Heading</Card.Heading>
    <Card.Description>
      Everything you need to know about getting started with GitHub Actions.
    </Card.Description>
  </Card>
  <Card linkHref="https://github.com">
    <Card.Heading>Heading</Card.Heading>
    <Card.Description>
      Everything you need to know about getting started with GitHub Actions.
    </Card.Description>
  </Card>
</Stack>
```

## Component props

### Card

| name       | type     | default     | required | description                             |
| ---------- | -------- | ----------- | -------- | --------------------------------------- |
| `linkHref` | `string` |             | `true`   | URL to the card content                 |
| `link`     | `string` | `Read more` | `false`  | Label of link at the bottom of the Card |

### Card.Heading

| name | type                         | default | required | description |
| ---- | ---------------------------- | ------- | -------- | ----------- |
| `as` | `h2`, `h3`, `h4`, `h5`, `h6` | `h3`    | `false`  |             |
