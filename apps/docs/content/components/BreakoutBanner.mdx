## Examples

### Default

```jsx live
<BreakoutBanner>
  <BreakoutBanner.Heading>
    Where the most ambitious teams build great things
  </BreakoutBanner.Heading>
  <BreakoutBanner.LinkGroup>
    <Link href="#">Primary action</Link>
  </BreakoutBanner.LinkGroup>
</BreakoutBanner>
```

### Optional description

### Alignment

The content alignment can be changed using the `align` prop on the root `BreakoutBanner`. This can be configured to either `start` or `center`.

```jsx live
<BreakoutBanner align="center">
  <BreakoutBanner.Heading>
    Where the most ambitious teams build great things
  </BreakoutBanner.Heading>
  <BreakoutBanner.Description>
    Lorem ipsum dolor sit amet, consectetur adipiscing elit. In sapien sit
    ullamcorper id. Aliquam luctus sed turpis felis nam pulvinar risus
    elementum.
  </BreakoutBanner.Description>
  <BreakoutBanner.LinkGroup>
    <Link href="#">Primary action</Link>
    <Link href="#">Secondary action</Link>
  </BreakoutBanner.LinkGroup>
</BreakoutBanner>
```

## Component props

### BreakoutBanner <Label>Required</Label>

| name                      | type                                                                                                       | default     | required | description                                                      |
| ------------------------- | ---------------------------------------------------------------------------------------------------------- | ----------- | -------- | ---------------------------------------------------------------- |
| `children`                | `ReactNode`, `ReactNode[]`                                                                                 | `undefined` | `true`   | Content to include within the banner component                   |
| `align`                   | <PropTableValues values={[`start`, `center`]} addLineBreaks commaSeparated />                              | `'start'`   | `true`   | The horizontal positioning of content wihin the banner component |
| `backgroundColor`         | <PropTableValues values={['default', 'subtle', 'string', `ResponsiveMap`]} addLineBreaks commaSeparated /> | `'subtle'`  | `false`  | Optional, custom background color                                |
| `backgroundImageSrc`      | <PropTableValues values={['string' , `ResponsiveMap`]} addLineBreaks commaSeparated />                     | `undefined` | `false`  | Optional, custom background image                                |
| `backgroundImagePosition` | <PropTableValues values={['string' , `ResponsiveMap`]} addLineBreaks commaSeparated/>                      | `'center'`  | `false`  | Optional, custom background position                             |
| `backgroundImageSize`     | <PropTableValues values={['string' , `ResponsiveMap`]} addLineBreaks commaSeparated />                     | `'cover'`   | `false`  | Optional, custom background position size                        |
| `leadingVisual`           | `ReactElement`, `Icon`                                                                                     | `undefined` | `false`  | An optional leading visual that appears before the heading       |

### BreakoutBanner.Heading <Label>Required</Label>

| name       | type                                                    | default     | required | description                                                           |
| ---------- | ------------------------------------------------------- | ----------- | -------- | --------------------------------------------------------------------- |
| `children` | `ReactNode`, `ReactNode[]`                              | `undefined` | `true`   | Content to be displayed inside the `BreakoutBanner.Heading` component |
| `as`       | <PropTableValues values={HeadingTags} addLineBreaks />  | `h3`        | `false`  | Heading level                                                         |
| `size`     | <PropTableValues values={HeadingSizes} addLineBreaks /> | `4`         | `false`  | Visual heading size, irrespective of level                            |

### BreakoutBanner.Description

Optional child that can be used to provide additional information.

| name       | type                       | default     | required | description                                         |
| ---------- | -------------------------- | ----------- | -------- | --------------------------------------------------- |
| `children` | `ReactNode`, `ReactNode[]` | `undefined` | `true`   | Content to be displayed inside the banner component |

### BreakoutBanner.LinkGroup

| name        | type                                                                                                  | default        | required | description                                  |
| ----------- | ----------------------------------------------------------------------------------------------------- | -------------- | -------- | -------------------------------------------- |
| `children`  | `Link`                                                                                                | undefined      | `true`   | The links to be displayed within the group   |
| `direction` | <PropTableValues values={['horizontal', 'vertical', `ResponsiveMap`]} addLineBreaks commaSeparated /> | `'horizontal'` | `false`  | Positioning of the links across x and y axis |

All components forward `id`, `className` and `ref` by default
