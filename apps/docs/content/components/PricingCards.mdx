## Component props

### PricingCards <Label>Required</Label>

| Name       | Type                |  Default  | required | Description                         |
| :--------- | :------------------ | :-------: | :------: | :---------------------------------- |
| `as`       | `section , div`     | `section` | `false`  | Applies the underlying HTML element |
| `children` | `PricingCards.Item` |           |  `true`  | Applies the underlying HTML element |
| `presentation` | `default, cards` |           |  `false`  | The presentation styling |

### PricingCards.Item <Label>Required</Label>

One item per pricing plan to be displayed. Maximum of 3 items.

| Name       | Type                                                                                                                                                                 |   Default   | required | Description              |
| :--------- | :------------------------------------------------------------------------------------------------------------------------------------------------------------------- | :---------: | :------: | :----------------------- |
| `children` | <PropTableValues values={['PricingCards.Heading', 'PricingCards.Label', 'PricingCards.Price', 'PricingCards.FeatureList', 'PricingCards.Footnote']} addLineBreaks /> | `undefined` |  `true`  | The contents of the item |

### PricingCards.Heading <Label>Required</Label>

| Name       | Type             |   Default   | required | Description                                       |
| :--------- | :--------------- | :---------: | :------: | :------------------------------------------------ |
| `as`       | `h3, h4, h5, h6` |    `h3`     | `false`  | Applies the underlying HTML heading level element |
| `children` | `ReactNode`      | `undefined` |  `true`  | The heading of the item                           |

### PricingCards.Label

| Name       | Type        |   Default   | required | Description           |
| :--------- | :---------- | :---------: | :------: | :-------------------- |
| `children` | `ReactNode` | `undefined` |  `true`  | The label of the item |

### PricingCards.Description

| Name       | Type        |   Default   | required | Description                |
| :--------- | :---------- | :---------: | :------: | :------------------------- |
| `children` | `ReactNode` | `undefined` |  `true`  | The desciption of the item |

### PricingCards.Price

| Name                     | Type                |   Default   | required | Description                         |
| :----------------------- | :------------------ | :---------: | :------: | :---------------------------------- |
| `children`               | `ReactNode`         | `undefined` |  `true`  | The price of the item               |
| `currencySymbol`         | `string`            |     `$`     | `false`  | The currency symbol                 |
| `currencySymbolPosition` | `leading, trailing` |  `leading`  | `false`  | The position of the currency symbol |
| `trailingText`           | `string`            |             | `false`  | The trailing text                   |

### PricingCards.FeatureList

| Name       | Type                                                                                       |   Default   | required | Description          |
| :--------- | :----------------------------------------------------------------------------------------- | :---------: | :------: | :------------------- |
| `children` | <PropTableValues values={['PricingCards.Heading', 'PricingCards.Item', ]} addLineBreaks /> | `undefined` |  `true`  | The list of features |

### PricingCards.FeatureListHeading

| Name       | Type         |   Default   | required | Description                                       |
| :--------- | :----------- | :---------: | :------: | :------------------------------------------------ |
| `as`       | `h4, h5, h6` |    `h4`     | `false`  | Applies the underlying HTML heading level element |
| `children` | `ReactNode`  | `undefined` |  `true`  | The heading of the feature list set               |

### PricingCards.FeatureListItem

| Name       | Type                 |   Default   | required | Description                         |
| :--------- | :------------------- | :---------: | :------: | :---------------------------------- |
| `children` | `ReactNode`          | `undefined` |  `true`  | The item of the feature list set    |
| `variant`  | `included, excluded` | `included`  | `false`  | The variant of the feature list set |

### PricingCards.Footnote

| Name       | Type        |   Default   | required | Description              |
| :--------- | :---------- | :---------: | :------: | :----------------------- |
| `children` | `ReactNode` | `undefined` |  `true`  | The footnote of the item |
