import {Label} from '@primer/react'

```js
import {Avatar, Stack} from '@primer/react-brand'
```

## Examples

### Default

```jsx live
<Avatar src="https://i.pravatar.cc/160?u=80" alt="A random avatar picture" />
```

### Sizes

The size of the avatar can be controlled by the `size` prop.

```jsx live
<Stack
  direction="horizontal"
  alignItems="center"
  justifyContent="center"
  gap="spacious"
  style={{height: 200}}
>
  <Avatar
    size={32}
    src="https://i.pravatar.cc/160?u=80"
    alt="A random avatar picture at 32px"
  />
  <Avatar
    size={40}
    src="https://i.pravatar.cc/160?u=80"
    alt="A random avatar picture at 40px"
  />
  <Avatar
    size={48}
    src="https://i.pravatar.cc/160?u=80"
    alt="A random avatar picture at 48px"
  />
  <Avatar
    size={64}
    src="https://i.pravatar.cc/160?u=80"
    alt="A random avatar picture at 64px"
  />
  <Avatar
    size={80}
    src="https://i.pravatar.cc/160?u=80"
    alt="A random avatar picture at 80px"
  />
</Stack>
```

### Shapes

The shape of the avatar can be controlled by the `shape` prop.

```jsx live
<Stack
  direction="horizontal"
  alignItems="center"
  justifyContent="center"
  gap="spacious"
  style={{height: 200}}
>
  <Avatar
    size={64}
    shape="circle"
    src="https://i.pravatar.cc/160?u=80"
    alt="A random avatar picture at 64px"
  />
  <Avatar
    size={64}
    shape="square"
    src="https://i.pravatar.cc/160?u=80"
    alt="A random avatar picture at 64px"
  />
</Stack>
```

## Component props

### Avatar <Label>Required</Label>

| name        | type                         | default  | required | description                       |
| ----------- | ---------------------------- | -------- | -------- | --------------------------------- |
| `alt`       | `string`                     |          | `true`   | The alt text of the image         |
| `className` | `string`                     |          | `false`  | Avatar custom class               |
| `size`      | `32`, `40`, `48`, `64`, `80` | `40`     | `false`  | The size (in pixels) of the image |
| `shape`     | `circle`, `square`           | `circle` | `false`  | The shape of the image            |
| `src`       | `string`                     |          | `true`   | A url to the person's image       |
