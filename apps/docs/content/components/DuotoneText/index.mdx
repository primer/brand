## Usage

Use duotone text to create a more dynamic reading flow by emphasizing words at the beggining of a text. Duotone text is a purely visual treatment and should not be used to convey semantic emphasis.

<DoDontContainer>
  <Do>
    <img alt="" src={doEmphasis} />
    <Caption>Only emphasize the first few words of a text.</Caption>
  </Do>
  <Dont>
    <img alt="" src={dontEmphasis} />
    <Caption>Don't emphasize other parts of the text.</Caption>
  </Dont>
</DoDontContainer>
