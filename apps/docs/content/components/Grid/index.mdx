## Usage

Use the grid component to define a layout that maintains consistent element positioning and sizing across views. Use the grid component to add guides for scale and placement and provide consistency across the product.

The grid component has 12 columns and uses vertical and horizontal gaps to separate content units consistently in all modules. Use the grid to group related content units. E.g. A group of [cards](/components/Card) or [pillars](/components/Pillar).
