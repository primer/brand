## Component props

### RiverAccordion <Label>Required</Label>

| Name       | Type                                                                                                          |  Default  | Description                                                            |
| :--------- | :------------------------------------------------------------------------------------------------------------ | :-------: | :--------------------------------------------------------------------- |
| `children` | <PropTableValues values={[<a href="#riveraccordionitem-required">RiverAccordion.Item[]</a>]} addLineBreaks /> |           | Multiple RiverAccordion.Item components                                |
| `align`    | <PropTableValues values={['start', 'end']} addLineBreaks />                                                   | `'start'` | Controls the position of the visuals relative to the accordion content |

`RiverAccordion` extends the HTML `div` element and supports all `div` props.

### RiverAccordion.Item <Label>Required</Label>

| Name       | Type                                                                                                                                                                                                                                                       | Default | Description                              |
| :--------- | :--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | :-----: | :--------------------------------------- |
| `children` | <PropTableValues values={[<a href="#riveraccordionheading-required">RiverAccordion.Heading</a>,<a href="#riveraccordioncontent-required">RiverAccordion.Content</a>, <a href="#riveraccordionvisual-required">RiverAccordion.Visual</a>]} addLineBreaks /> |         | Must include all three listed components |

`RiverAccordion.Item` extends the HTML `div` element and supports all `div` props.

### RiverAccordion.Heading <Label>Required</Label>

| Name       | Type     | Default | Description  |
| :--------- | :------- | :-----: | :----------- |
| `children` | `string` |         | Heading text |

`RiverAccordion.Heading` extends the [`Heading`](/components/Heading) component and supports all `Heading` props.

### RiverAccordion.Content <Label>Required</Label>

| Name       | Type                   | Default | Description        |
| :--------- | :--------------------- | :-----: | :----------------- |
| `children` | `React.ReactElement[]` |         | Content components |

`RiverAccordion.Content` extends the HTML `div` element and supports all `div` props.

### RiverAccordion.Visual <Label>Required</Label>

| Name       | Type                 | Default | Description                  |
| :--------- | :------------------- | :-----: | :--------------------------- |
| `children` | `React.ReactElement` |         | Visual content (e.g., image) |

`RiverAccordion.Visual` extends the HTML `div` element and supports all `div` props.
