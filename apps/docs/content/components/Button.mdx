```js
import {Button} from '@primer/react-brand'
```

## Examples

### Default

This is the default variant for the Button component. It corresponds to the `secondary` variant appearance.

```jsx live
<Button>Default</Button>
```

### Primary

The `primary` variant of Button indicates that the action is of high priority, and should be used sparingly.

```jsx live
<Button variant="primary">Primary</Button>
```

### Subtle

The `subtle` variant of Button indicates that the action is a low priority one.

```jsx live
<Button variant="subtle">Primary</Button>
```

### Sizes

```jsx live
<Stack direction="vertical" alignItems="flex-start">
  <Button size="medium">Register now</Button>
  <Button size="large">Register now</Button>
</Stack>
```

### Removing the arrow

The animated arrow indicator can be removed using the `hasArrow` prop.

```jsx live
<Button hasArrow={false}>No arrow</Button>
```

### Appending an icon

We can place an icon inside the Button in either the leading or the trailing position to enhance the visual context. It is recommended to use an [Octicon](https://primer.style/octicons) here.

```jsx live
<Stack direction="vertical" alignItems="flex-start">
  <Button leadingVisual={SearchIcon}>Search</Button>
  <Button trailingVisual={SearchIcon}>Search</Button>
  <Button leadingVisual={SearchIcon} trailingVisual={CheckIcon}>
    Search
  </Button>
</Stack>
```

### Polymorphism

The `Button` component can render as a `button` or `a` HTML element. By default, it will render as a `button`.

```jsx live
<Button as="a" href="https://github.com">
  Register now
</Button>
```

## Component props

### Button

| Name             | Type                                                      |    Default    | Description                                        |
| :--------------- | :-------------------------------------------------------- | :-----------: | :------------------------------------------------- |
| `leadingVisual`  | `ReactElement`                                            |  `undefined`  | The leading icon appears before the button content |
| `trailingVisual` | `ReactElement`                                            |  `undefined`  | The trailing icon appears after the button content |
| `variants`       | <PropTableValues values={ButtonVariants} addLineBreaks /> | `'secondary'` | The styling variations available in Button         |
| `size`           | <PropTableValues values={ButtonSizes} addLineBreaks />    |  `'medium'`   | The size variations available in Button            |
| `hasArrow`       | `boolean`                                                 |    `true`     | A flag to show/hide the arrow icon                 |
