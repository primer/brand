## Component props

### Prose

| Name              | Type                                   |  Default  | Description                                                        |
| :---------------- | :------------------------------------- | :-------: | :----------------------------------------------------------------- |
| `className`       | `string`                               |           | Sets a custom class on the root element                            |
| `rawHtmlMarkup`   | `React.ReactNode`, `React.ReactNode[]` | `default` | Content to be displayed within the Prose component                 |
| `enableFullWidth` | `boolean`                              |  `false`  | A toggle to remove the default max-width if required by the design |
